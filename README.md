Fyber Ruby Developer Challenge
===============

Web app to search and display offers using fyber API.

## Setup

Create a `.env` file with following keys with corresponding values:
```yml
FYBER_APPID:
FYBER_DEVICE_ID:
FYBER_LOCALE:
FYBER_IP:
FYBER_API_KEY:
```


Install bundle

`bundle`

## Run

`bundle exec dotenv rails s`

## Test

`bundle exec rake spec`
